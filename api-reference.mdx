## Overview

Complete JSON-RPC API reference for Paxeer Network. All standard Ethereum methods are supported.

<Info>
**Endpoint:** `https://public-rpc.paxeer.app/rpc`
</Info>

## Authentication

No authentication required. All API methods are publicly accessible.

## Request Format

All requests follow the JSON-RPC 2.0 specification:

```json
{
  "jsonrpc": "2.0",
  "method": "method_name",
  "params": [],
  "id": 1
}
```

## Available Methods

### eth_blockNumber

<ParamField path="method" type="string" required>
  eth_blockNumber
</ParamField>

Returns the number of most recent block.

**Parameters:** None

**Returns:** `QUANTITY` - Integer of the current block number

<CodeGroup>

```json Request
{
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": [],
  "id": 1
}
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x5daf3b"
}
```

</CodeGroup>

### eth_getBalance

Returns the balance of the account of given address.

**Parameters:**

<ParamField path="address" type="DATA" required>
  20 Bytes - Address to check for balance
</ParamField>

<ParamField path="block" type="QUANTITY|TAG" required>
  Integer block number, or the string "latest", "earliest" or "pending"
</ParamField>

**Returns:** `QUANTITY` - Integer of the current balance in wei

<CodeGroup>

```json Request
{
  "jsonrpc": "2.0",
  "method": "eth_getBalance",
  "params": [
    "0x407d73d8a49eeb85d32cf465507dd71d507100c1",
    "latest"
  ],
  "id": 1
}
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x0234c8a3397aab58"
}
```

</CodeGroup>

### eth_sendTransaction

Creates new message call transaction or a contract creation.

**Parameters:**

<ParamField path="transaction" type="Object" required>
  Transaction object:
  - `from`: Address - The address the transaction is sent from
  - `to`: Address - (optional) The address the transaction is directed to
  - `gas`: QUANTITY - (optional) Gas provided for the transaction execution
  - `gasPrice`: QUANTITY - (optional) Gas price
  - `value`: QUANTITY - (optional) Value sent with this transaction
  - `data`: DATA - (optional) Compiled contract code or hash of invoked method
</ParamField>

**Returns:** `DATA`, 32 Bytes - The transaction hash

<CodeGroup>

```json Request
{
  "jsonrpc": "2.0",
  "method": "eth_sendTransaction",
  "params": [{
    "from": "0xb60e8dd61c5d32be8058bb8eb970870f07233155",
    "to": "0xd46e8dd67c5d32be8058bb8eb970870f07244567",
    "gas": "0x76c0",
    "gasPrice": "0x9184e72a000",
    "value": "0x9184e72a",
    "data": "0x..."
  }],
  "id": 1
}
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d5145a99d05921026d1527331"
}
```

</CodeGroup>

### eth_call

Executes a new message call immediately without creating a transaction.

**Parameters:**

<ParamField path="transaction" type="Object" required>
  Transaction call object
</ParamField>

<ParamField path="block" type="QUANTITY|TAG" required>
  Integer block number, or the string "latest", "earliest" or "pending"
</ParamField>

**Returns:** `DATA` - The return value of executed contract

<CodeGroup>

```json Request
{
  "jsonrpc": "2.0",
  "method": "eth_call",
  "params": [{
    "to": "0xd46e8dd67c5d32be8058bb8eb970870f07244567",
    "data": "0x..."
  }, "latest"],
  "id": 1
}
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x"
}
```

</CodeGroup>

    "value": "0x9184e72a"

## Code Examples

### cURL

```bash
curl -X POST https://public-rpc.paxeer.app/rpc \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_blockNumber",
    "params": [],
    "id": 1
  }'
```

### JavaScript/TypeScript

```javascript
const response = await fetch('https://public-rpc.paxeer.app/rpc', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    jsonrpc: '2.0',
    method: 'eth_blockNumber',
    params: [],
    id: 1,
  }),
});

const data = await response.json();
console.log(data.result);
```

### Python

```python
import requests

url = 'https://public-rpc.paxeer.app/rpc'
payload = {
    'jsonrpc': '2.0',
    'method': 'eth_blockNumber',
    'params': [],
    'id': 1
}

response = requests.post(url, json=payload)
print(response.json())
```

## Rate Limits

Currently, there are no rate limits on the public RPC endpoint. However, please be considerate and avoid excessive requests.

## Error Codes

| Code | Message | Description |
|------|---------|-------------|
| -32700 | Parse error | Invalid JSON |
| -32600 | Invalid request | JSON is not a valid request object |
| -32601 | Method not found | Method does not exist |
| -32602 | Invalid params | Invalid method parameters |
| -32603 | Internal error | Internal JSON-RPC error |
