## Network Details

Connect to Paxeer Network with these details:

```json Network Configuration
{
  "networkName": "Paxeer Network",
  "rpcUrl": "https://public-rpc.paxeer.app/rpc",
  "chainId": 229,
  "currencySymbol": "PAX",
  "blockExplorer": "https://scan.paxeer.app"
}
```
